Projects

Commit to the Lord whatever you do, and He will establish your plans. — Proverbs 16:3

  • Herd Genetics

    Herd Genetics is a full stack web simulation used by Penn State and Cornell University. Dedicated to teaching genetics, it simulates breeding programs, focusing on PTAs, genetic recessives and inbreeding coefficients. Herd Genetics aims for accuracy of PTA/Trait correlations and trends. It is a classroom based system targeting group learning with every simulation belonging to a class.

  • videre.nvim

    Many editors have the option to view JSON & YAML files as a graph. Neovim, with a terminal interface, does not have this luxury. While one can't create an interface like JSON Crack, it is possible to build a similar JSON explorer using Neovim's terminal interface. Videre does just that.

  • AP CSP - CSP Pseudocode Interpreter

    College Board's Advanced Placement Computer Science Principals class uses pseudocode instead of an actual language on the AP test. It is important though to understand this pseudocode. Therefore, I implemented a simple pseudocode interpreter, so you can practice in the language you will be tested on.

  • PyPi: inbreeding-calculator

    inbreeding-calculator is a PyPi python package for calculating the inbreeding coefficient of a pedigrees in Python.

  • Farmle

    Do you think you need Wordle for the farm? Yes, you do! Here it is: Farmle.

  • TermsLang

    TermsLang is a custom programming language built using Rust. Perhaps one of the least practical of these projects, its sole purpose is to be an advanced learning experience and an example to others of what is possible using Rust.
